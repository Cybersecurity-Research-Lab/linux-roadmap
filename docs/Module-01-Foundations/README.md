# 🟢 Module 01 – Linux Foundations

> *"Every cybersecurity professional starts by understanding the systems they aim to secure. Linux is one of those foundations."*

Welcome to the first module of the **Linux Roadmap**.

This module introduces the core Linux concepts every learner should master before moving on to system administration, networking, automation, and cybersecurity.

The goal is not to memorize commands, but to understand how Linux works and build a solid foundation that will support every topic that follows.

---

# 🎯 Learning Outcomes

By completing this module, you will be able to:

* Explain what Linux is and why it is widely used.
* Install Linux in a virtual machine or on physical hardware.
* Navigate confidently using the Linux terminal.
* Understand the Linux file system hierarchy.
* Manage files and directories.
* Use essential Linux commands.
* Work with text editors such as Nano and Vim.

To get the most out of this module:

* Perform every command yourself.
