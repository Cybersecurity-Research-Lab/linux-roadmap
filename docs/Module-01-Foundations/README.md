# 🟢 Module 01 – Linux Foundations

Welcome to the first module of the Linux Roadmap.

This module introduces the fundamental concepts every Linux user should understand before moving on to system administration, networking, automation, and cybersecurity.

---

## 🎯 Skills You'll Learn

By completing this module, you will be able to:

- Understand what Linux is
- Install Linux
- Navigate the terminal
- Understand the Linux file system
- Manage files and directories
- Use basic Linux commands
- Work with text editors
